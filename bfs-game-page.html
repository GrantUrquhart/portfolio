Breadth First Search Game

Github repository link: https://github.com/GrantUrquhart/breadth-first-search-game
Overview:

In this project, I developed a program to solve a board game with a 2x4 grid using the breadth-first search (BFS) algorithm. The goal is to find the shortest path from a randomly numbered starting board to the winning configuration, where the numbers are in order from 1 to 8.

Key Features:
  • Implemented the BFS algorithm to explore all possible moves from the starting state and find the quickest path to victory.
  • Designed a data structure to represent the game board and efficiently track the state of the game.
  • Ensured that the program can handle starting boards that require at least 7 moves to reach the winning configuration.
Technologies Used:
  • Java for the implementation of the BFS algorithm and game logic.
  • Custom data structures for efficient state management and move tracking.
Challenges Faced:
  • State Space Explosion: Managing the exponential growth of possible game states as the BFS algorithm explores moves was a significant challenge. I addressed this by optimizing the data structures used for storing and retrieving game states.
  • Efficient Move Generation: Developing an efficient method to generate and evaluate possible moves from each game state was crucial for performance. I implemented a move generation algorithm that minimized unnecessary computations.
What I Learned:
  • Algorithmic Thinking: This project deepened my understanding of the BFS algorithm and its application in solving real-world problems.
  • Data Structure Design: I gained valuable experience in designing data structures that are optimized for specific algorithmic requirements.
  • Performance Optimization: The project taught me the importance of performance optimization in algorithm implementation, especially in scenarios with a large state space.
Conclusion:

The Board Game Solver project was an enriching experience that allowed me to apply algorithmic concepts to a practical problem. It not only honed my skills in algorithm design and implementation but also provided insights into the challenges of optimizing algorithms for performance and scalability.
